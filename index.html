📊 Project Summary

This project explores reintegration outcomes using a custom-built SQL Server database and interactive Power BI dashboards. It covers economic, social, and psychosocial metrics.

🗃️ SQL Server Database Design

I designed and normalized a 15-table relational database with foreign key constraints, covering health, education, migration, NFI distribution, and reintegration outcomes.

🧠 Key SQL Queries

  • Returnee sustainability status based on threshold scoring
  • Cross-sectional analysis by gender and country of return
  • Dashboard-ready views for Power BI integration
